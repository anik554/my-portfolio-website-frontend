## Project Overview
This project is a modern web application built with **Next.js** (App Router).  
It demonstrates server-side rendering, API integration, dynamic routing, authentication, and responsive design.  

**Live Demo:** [https://portfolio-frontend-snowy-ten.vercel.app/]

---

## Features
- Server-Side Rendering (SSR) and Client-Side Rendering (CSR)
- Dynamic routing and parameterized pages
- Authentication using JWT / NextAuth.js
- Form handling with React Hook Form & Zod validation
- Responsive UI with Tailwind CSS
- API integration with a backend (Express/Node.js)
- Toast notifications for user feedback
- Environment-based configuration (local & production)

---

## Technology Stack
- **Frontend:** Next.js (v14+), React, TypeScript, Tailwind CSS
- **Backend (optional):** Express.js, Node.js
- **Authentication:** JWT / NextAuth.js
- **State Management:** React Context API or Redux Toolkit
- **Forms & Validation:** React Hook Form, Zod
- **UI Components:** Shadcn/UI, Tailwind CSS
- **Deployment:** Vercel
